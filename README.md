# Niche Validation Test - Multi-Tenant Landing Pages

## Purpose

Test demand for AI Content + Twitter Automation across 4 vertical niches:

1. **AgentOS** - For Real Estate Agents
2. **LawyerContent** - For Attorneys  
3. **ClinicGrow** - For Healthcare Providers
4. **DealerBoost** - For Car Dealerships

## Strategy

- Run $10/day Google Ads for each ($40/day total)
- Collect pre-orders ($49 each)
- See which niche has most demand
- Build the winner!

## Metrics to Track

- Ad clicks
- Cost per click
- Email signups
- **Pre-orders (most important!)**
- Cost per pre-order

## Timeline

- Week 1: Build 4 landing pages
- Week 2-3: Run Google Ads test ($280 total)
- Week 4: Analyze results, build winner

## Tech Stack

- Vite + React
- GitHub Pages deployment
- Google Analytics tracking
- Stripe/Gumroad for pre-orders

## Deploy

```bash
npm install
npm run build
npm run deploy
```

Deploys to GitHub Pages automatically!
